# Ecosystem Readiness Report

## Ecosystem readiness
A developer can define a deterministic Rustrig package, certify it, publish a registry artifact, and make it visible to Studio.

## Registry readiness
Registry artifacts include manifest.toml, metadata.json, package.bin, hash.sha256, and signature.bin.

## Studio readiness
Creator workflow supports browsing marketplace packages, installing Rustrigs, composing gameplay, running simulation through existing runtime surfaces, and publishing games.

## Security guarantees
Replay remains deterministic and runtime authority is not granted to marketplace content.

## Known limitations
Arena Vanguard uses local certified packages as viability proof while external registry hosting remains out of scope.
