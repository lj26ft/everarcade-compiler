# Game Creation Report

## Time-to-first-playable-game readiness

The creator workflow prioritizes immediate playability: install Studio, create a project, choose a template, generate a game, press play, run multiplayer, publish, and let players join.

## Required templates

- Arena
- Action RPG
- Civilization
- RTS
- Survival
- Sandbox
- Dungeon Crawler
- MMO Prototype

## Play modes

Single Player, Multiplayer, Persistent World, and Replay Mode are represented as one switchable Studio play workflow.
