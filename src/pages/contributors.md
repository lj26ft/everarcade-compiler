# Contributors

EverArcade is organized as a source-controlled runtime, SDK, tooling, and documentation project.

## Repository orientation

Start with the [repository map](/docs/architecture/repository-map) and the [human-readable repository map](/docs/architecture/human-readable-repo-map) to find runtime, SDK, frontend, operator, and documentation areas.

## How to contribute

1. Read the [contributor guide](/docs/contributor-guide/).
2. Build or validate only the area you changed.
3. Keep public documentation world-first and beginner friendly.
4. Submit improvements with clear tests, docs, and migration notes.

## Local development

Use the [developer quickstart](/docs/developer/quickstart) for world-building flows and the repository docs for component-specific commands.
