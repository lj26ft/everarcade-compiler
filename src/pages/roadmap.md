# Roadmap

EverArcade progresses through capability milestones rather than date promises.

## Milestone progression

1. **Current Platform** — source-controlled runtime, SDK, templates, operator notes, and documentation scaffolds.
2. **Public Testnet** — public participation path for builders, operators, and reviewers.
3. **Open Source Release** — repository hygiene, contributor onboarding, release integrity, and governance conventions.
4. **World Templates** — reusable starting points for arenas, civilizations, RPGs, sandboxes, trading, and additional genres.
5. **RustRigs Expansion** — common gameplay modules for inventory, combat, markets, quests, and governance.
6. **Federation Growth** — more operator patterns for recoverable, independently verified continuity.
7. **Marketplace Systems** — discovery, packaging, and exchange systems for worlds, modules, and services.
8. **Governance Evolution** — community processes for standards, review, and ecosystem coordination.
