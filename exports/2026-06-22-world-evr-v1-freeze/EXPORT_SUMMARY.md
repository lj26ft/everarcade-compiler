EverArcade Compiler is the source-of-truth repository for runtime, specification, verifier, registry, marketplace, treasury, proof, and CLI implementation state. The current export captures the compiler reality after the `world.evr` Package Spec V1 freeze, independent verifier completion, replay / restore / migration proof completion, World Registry implementation, Capability Marketplace implementation, Treasury implementation, Founding World infrastructure, and frontend extraction.
