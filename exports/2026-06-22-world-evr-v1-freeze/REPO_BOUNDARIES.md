# Repository Boundaries

## everarcade-compiler

Owns:

- runtime
- specs
- proofs
- registry logic
- marketplace logic
- treasury logic
- CLI

Does NOT own:

- public website
- docs deployment
- vision site
- portal deployment

## everarcade-frontend

Owns:

- `everarcade.games`
- `docs.everarcade.games`
- `vision.everarcade.games`
- `portal.everarcade.games`

## everarcade-hq

Owns:

- strategy
- roadmap
- narrative
- outreach
- operating memory
