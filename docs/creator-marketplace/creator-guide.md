# Creator Marketplace Guide

EverArcade Creator Marketplace v0.1 is a creator-to-creator publishing layer for reusable game-building components. It is not a player marketplace, does not custody keys or funds, and does not perform production billing.

Creators register a creator ID, publisher ID, profile ID, status, and registration epoch. Registry events support register, update, suspend, and recover, with every state represented by deterministic records and roots.

Supported publication families are templates, assets, quest packs, NPC packs, UI packs, economy packs, civilization packs, and code modules. Every object carries a publication ID, creator ID, type, version, publisher, history, and status.
