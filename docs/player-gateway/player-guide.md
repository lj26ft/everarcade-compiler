# Player Guide

The EverArcade Player Gateway lets players discover games, create a portable profile, join worlds, and return to recent sessions without needing to understand runtime, settlement, or replay infrastructure.

## Player Flow

1. Browse featured or published games.
2. Register or recover a player profile.
3. Create or restore a character for a game.
4. Join a game, world, or civilization session.
5. View inventory and marketplace activity.
6. Observe civilization events and economy surfaces.

The gateway records these actions for auditability, but the authoritative game runtime remains outside the gateway.
