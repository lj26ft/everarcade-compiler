# Operators

Operators keep verified worlds online. Your job is not only to run servers; it is to preserve world continuity.

## Operator responsibilities

- Publish world.evr operator metadata.
- Retain journals, receipts, checkpoints, restore bundles, and migration bundles.
- Support independent replay, restore, and migration verification.
- Expose proof status to the Registry so players and contributors know what they are joining.
- Execute treasury policy only through auditable world processes.
