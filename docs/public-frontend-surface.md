# Public Frontend Surface

Public websites, `docs.everarcade.games`, `vision.everarcade.games`, and the world portal live in `everarcade-frontend`.

Use these destinations for public presentation work:

- `everarcade-frontend` — repository for public websites, docs presentation, vision pages, and portal UI.
- `https://docs.everarcade.games` — public documentation site.
- `https://vision.everarcade.games` — public vision site.

This repository keeps compiler-owned implementation material only: deterministic runtime, proof/oracle specs, world packaging, CLI, registry primitives, capability marketplace primitives, treasury execution, and runtime verification.

For migration details, see `public-surface-export/README.md` and `public-surface-export/IMPORT_INSTRUCTIONS.md`.
