# EverArcade v0.1.0 Release Scope

## Included in v0.1.0
- Linux operator tarball (`everarcade-host-operator-v*.tar.gz`).
- `install.sh` and `uninstall.sh` flows from an extracted tarball.
- CLI operator lifecycle commands: `init`, `run`, `verify`, `status`, `debug`, `doctor`.
- `deploy-proof` dry-run flow for local deployment proof generation.
- Deterministic fixture package generation via `generate-fixture`.
- Release validation pipeline (`scripts/release_validate.sh`, `scripts/install_smoke.sh`).

## Explicitly excluded from v0.1.0
- Production XRPL live submission as a guaranteed-ready path.
- Production Evernode deployment as a guaranteed-ready path.
- Production game SDK.
- Production peer security hardening.

## Release contract
v0.1.0 is an operator-focused local deployment candidate: installable, testable, and auditable on Linux; not yet a full production gaming platform.
