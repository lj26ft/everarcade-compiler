# EverArcade Documentation

EverArcade is a sovereign game runtime for persistent worlds.

This documentation portal is the canonical public onboarding experience. Instead of starting with a whitepaper, choose the path that matches what you want to do.

## Choose your path

- [Developers](./developers/) create worlds, World Contracts, templates, and runtime packages.
- [Operators](./operators/) run worlds, retain history, verify execution, and recover continuity.
- [Players](./players/) learn why worlds persist, what ownership means, and how communities accumulate history.
- [Contributors](./contributor-guide/) improve the repository, documentation, SDK, and runtime.
