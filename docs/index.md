# EverArcade Documentation

EverArcade builds **world.evr**: infrastructure for portable, verifiable, persistent worlds.

A `world.evr` is a world package with the history, proofs, operator metadata, capabilities, treasury policy, and restore or migration evidence needed to move a world without losing continuity. EverArcade is the organization and ecosystem around that primitive.

## What EverArcade is

EverArcade is verified world infrastructure. It connects World Git, replay proofs, restore and migration bundles, registry discovery, a capability marketplace, and treasury execution so worlds can outlive a single server, operator, client, or release.
