# Evernode Integration

Evernode integration describes how operator infrastructure can host EverArcade runtime workloads.

The public message is simple: worlds can be operated by infrastructure providers while preserving package portability, replay material, and recovery procedures.

See [Evernode deployment](../evernode-deployment), [Evernode runtime model](../evernode-runtime-model), and [live deployment notes](../deployment/live-evernode-world-deployment-v0.1).
