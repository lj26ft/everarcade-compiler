# Runtime Protocol Naming Law

Canonical names are frozen for v0.1.0:
- **Root** = deterministic aggregate hash root (`RuntimeValidationRoot`, `WorldContinuityRoot`).
- **Hash** = hash of one artifact or request payload.
- **Checkpoint** = durable cut in runtime progression.
- **Snapshot** = materialized recoverable world image.
- **Window** = bounded runtime/event/replay interval.
- **Cursor** = forward-only deterministic progression pointer.
- **Witness** = deterministic evidence artifact.
- **Proof** = verification bundle that can validate witness or state.
