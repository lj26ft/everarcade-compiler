# Game Discovery Search Guide

Search supports three deterministic modes:

- Keyword search: normalized terms match indexed game names, categories, and tags.
- Category search: players filter by Arena, RPG, Trading, Civilization, Sandbox, Strategy, Action, or Adventure.
- Tag search: players filter by creator-supplied tags.

Search results use deterministic ordering by status, category, tag match, and game ID. The v0.1 layer does not use ML ranking, paid advertising, or sponsorship priority.
