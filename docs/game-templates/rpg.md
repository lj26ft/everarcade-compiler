# RPG Guide

## Create

```bash
everarcade new my-rpg --template rpg
```

## Build

```bash
everarcade build --project my-rpg
```

## Deploy

```bash
everarcade deploy --project my-rpg --target local
```

## Customize

Start with `src/game.js`, then add systems, assets, logic, and content without changing the template foundation.

## Included Systems
- Characters
- Equipment
- Inventory
- NPCs
- Quest Framework
- Regions

## Demonstrates
- Persistence
- Progression
- Marketplace Integration
