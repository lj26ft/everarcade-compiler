# Arena Guide

## Create

```bash
everarcade new my-arena --template arena
```

## Build

```bash
everarcade build --project my-arena
```

## Deploy

```bash
everarcade deploy --project my-arena --target local
```

## Customize

Start with `src/game.js`, then add systems, assets, logic, and content without changing the template foundation.

## Included Systems
- Players
- Combat
- Inventory
- Match Flow
- Score Tracking

## Demonstrates
- Renderer
- Physics
- Economy Rewards
