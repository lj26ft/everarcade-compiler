# Open Source Contributor Guide

EverArcade welcomes contributors who improve developer onboarding, operator clarity, verifier confidence, and public documentation. Prefer targeted validation over broad rebuilds.

## Developer

- **Where to start:** README, First World guide, Creator SDK docs, templates.
- **Recommended docs:** `docs/first-world.md`, `docs/creator-sdk-v1.1.md`, `docs/world-template-library-v1.md`, `docs/world-package-spec-v1.md`.
- **Suggested first contribution:** improve a template README, add a small Creator SDK validation, or clarify a World Package error message.

## Operator

- **Where to start:** runtime docs and recovery runbooks.
- **Recommended docs:** `docs/runtime/index.md`, `docs/operator-recovery.md`, `docs/runbooks/checkpoint_restore.md`, `docs/runtime/world_restoration.md`.
- **Suggested first contribution:** improve a runbook, document an operational failure mode, or add an evidence-retention checklist.

## Verifier

- **Where to start:** verification overview, certification framework, replay docs.
- **Recommended docs:** `docs/open-source-launch/verification-overview.md`, `docs/verification-story-v1.md`, `docs/world-package-certification-framework-v1.md`, `docs/replay-verification.md`.
- **Suggested first contribution:** add a verifier checklist, improve invariant documentation, or expand a replay/restore example.

## Documentation Contributor

- **Where to start:** documentation index and documentation policy.
- **Recommended docs:** `docs/index.md`, `docs/DOCUMENTATION_POLICY.md`, `docs/open-source-launch/launch-overview.md`.
- **Suggested first contribution:** fix broken links, remove stale internal wording, or add glossary cross-links.

## Community Contributor

- **Where to start:** launch overview, why EverArcade, example gallery.
- **Recommended docs:** `docs/open-source-launch/why-everarcade.md`, `docs/open-source-launch/what-can-i-build.md`, `docs/open-source-launch/example-gallery.md`.
- **Suggested first contribution:** write a beginner explanation, test the First World path, or propose a community-friendly example world.
