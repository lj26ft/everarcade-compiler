# Launch Announcement Assets

## GitHub description

Open-source deterministic game runtime and creator tooling for packageable, replay-verifiable persistent worlds.

## GitHub topics

`game-runtime`, `deterministic-simulation`, `persistent-worlds`, `world-packages`, `replay-verification`, `rust`, `nodejs`, `creator-tools`, `open-source-games`, `evernode`, `xrpl`

## Short project summary

EverArcade lets developers create deterministic game worlds, package them as portable World Packages, run them locally, and verify execution with receipts, journals, checkpoints, replay evidence, and continuity roots.

## One-paragraph summary

EverArcade is an open-source platform for sovereign game worlds: developers define World Contracts, compose RustRig gameplay capabilities, produce portable World Packages, and generate evidence that operators and verifiers can inspect. The v0.1 launch focuses on local proof through the First World flow, template discovery, package creation, deterministic execution, replay/restore concepts, and contributor onboarding.

## Elevator pitch

EverArcade is for worlds that should be remembered, moved, and verified. Instead of treating a game server as the only source of truth, EverArcade packages the world, declares its rules, records execution evidence, and gives developers, operators, and verifiers a shared artifact to inspect.

EverArcade advances open infrastructure for persistent digital worlds by making world definitions, runtime artifacts, and verification evidence portable. The v0.1 repository demonstrates local creator onboarding, reusable templates, World Package certification concepts, deterministic execution evidence, and verifier-facing replay/restore/migration documentation.
