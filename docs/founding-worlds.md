# Founding Worlds

The Founding Worlds Program is the first public cohort of worlds proving `world.evr` in practice.

## Program

- Ten founding world slots.
- Public world profiles in the Registry portal.
- Help preparing replay, restore, migration, capability, treasury, and contributor metadata.
- Clear calls for players, operators, artists, designers, and protocol builders.

## Apply

Apply if you want to build a persistent world that can be played, verified, contributed to, and forked.

**CTA:** Apply for Founding Worlds and come build worlds with us.
