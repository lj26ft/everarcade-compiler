# EverArcade v0.1 Release Notes

## Release Status

EverArcade v0.1 is release certified for operator onboarding, developer onboarding, public demonstrations, reference world deployment, and open source launch preparation.

## What Exists

The v0.1 release includes certified artifacts and documentation for:

- Runtime
- Replay
- Federation
- Migration
- Restore
- Canonicalizer
- Proof Boundary
- World Contracts
- RustRigs
- World Packages
- Registry
- Reference World

## Release Certification

The release decision is recorded in:

- `docs/release/everarcade-v0.1-release-certification.md`
- `reports/release-certification/release-report.txt`

Final certification output:

```text
EVERARCADE V0.1 RELEASE CERTIFICATION: PASS
```

## Operator Outcome

An independent operator can complete the v0.1 workflow without reading internal implementation code:

1. Clone repository.
2. Build package.
3. Verify package.
4. Inspect certifications.
5. Deploy world.
6. Replay world.
7. Restore world.
8. Migrate world.
9. Verify roots.

## What Is Experimental

The following areas remain experimental, alpha, prototype, planned, or future work in v0.1:

- Renderer/history/federation domains remain scaffold-level runtime domains unless explicitly certified by a release report.
- Public federated registry hosting beyond the local reference registry is future work.
- Production-scale operator automation beyond documented scripts and runbooks is future work.
- Additional RustRig families beyond the certified v0.1 set are planned.
- Advanced marketplace, governance, and settlement integrations remain bounded by their respective certification artifacts and may require further certification before production use.
- UI and developer-experience surfaces outside the reference operator workflow may remain alpha or prototype depending on their local documentation.

## Release Gate

EverArcade v0.1 is the final release gate before:

- Open Source Launch
- Operator Onboarding
- Developer Onboarding
- Public Demonstrations
- Reference World Deployment
