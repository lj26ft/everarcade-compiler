# counter-world (v0.1 example)

Deterministic operator demo package flow:
1. `everarcade-host generate-fixture --output /tmp/counter-world.bin`
2. `everarcade-host run --package /tmp/counter-world.bin --state ~/.everarcade`
3. `everarcade-host verify --state ~/.everarcade`

This example intentionally uses the deterministic fixture pipeline as the first developer-facing demo for v0.1.0.
