# Settlement World Template

## Purpose

Economic and governance worlds.

## Demonstrates

Worlds as economies.

## Included RustRigs

- `market.trade`
- `market.list`
- `governance.vote`
- `governance.create_proposal`
- `structure.build`
- `inventory.transfer`

## Continuity

institutional: market history, ownership records, and enacted proposals remain durable

## Customize

1. Edit `world-contract.toml` authority rules and receipts.
2. Replace `genesis/state.toml` with project-specific starting state.
3. Update `projection/projection.toml` and assets for the desired player/operator view.
4. Build the World Package and submit it through certification level 3.
