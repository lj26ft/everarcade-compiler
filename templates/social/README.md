# Social World Template

## Purpose

Community and social worlds.

## Demonstrates

Worlds as communities.

## Included RustRigs

- `identity.join`
- `faction.create`
- `faction.join`
- `guild.create`
- `quest.start`
- `quest.complete`

## Continuity

social: membership, events, guild milestones, and completed quests are recorded

## Customize

1. Edit `world-contract.toml` authority rules and receipts.
2. Replace `genesis/state.toml` with project-specific starting state.
3. Update `projection/projection.toml` and assets for the desired player/operator view.
4. Build the World Package and submit it through certification level 2.
