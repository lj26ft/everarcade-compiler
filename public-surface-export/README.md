# EverArcade Public Surface Export

This bundle contains the public-facing website, docs, vision, portal, registry fixture, deployment-note, and X/social positioning surface exported from `everarcade-compiler` for import into `everarcade-frontend`.

`everarcade-compiler` remains the source of truth for deterministic runtime, proof/oracle specs, CLI, world packaging, registry primitives, capability marketplace primitives, treasury execution, and runtime verification.

## Bundle layout

- `docs-site/` — Docusaurus config, Vercel config, public docs snapshot, and launch/narrative pages.
- `vision-site/` — vision and Founding Worlds pages intended for `vision.everarcade.games`.
- `world-portal/` — Vite world portal app plus exported developer portal records for import/reference.
- `registry-fixtures/` — public testnet and registry fixture trees for frontend development.
- `deployment-notes/` — public deployment/testnet launch notes and domain/Vercel context.
- `x-positioning/` — X/social positioning copy.
- `INVENTORY.md` — source inventory captured before moving the surface.
- `IMPORT_INSTRUCTIONS.md` — import map, build commands, domains, and Vercel notes.
