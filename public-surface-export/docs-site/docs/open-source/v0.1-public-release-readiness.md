# EverArcade v0.1 Public Release Readiness

Core public message: **Build Worlds. Not Just Games.**

This page is the public-consumption checklist for EverArcade v0.1. It does not introduce protocol changes, certification frameworks, RustRigs, or specifications.

## Canonical demo world

Use `examples/reference-certified-world-v1/` as the v0.1 public demo package.

```bash
cd examples/reference-certified-world-v1
./operator/build-world-evr.sh .
./operator/verify.sh .
```

```text

```

## Fresh clone quick start validation

From a fresh machine with Git, Bash, tar, and sha256sum:

```bash
git clone <repo-url> everarcade-compiler
cd everarcade-compiler
bash examples/reference-certified-world-v1/operator/build-world-evr.sh examples/reference-certified-world-v1 /tmp/everarcade-v0.1-world.evr
tar -tzf /tmp/everarcade-v0.1-world.evr | LC_ALL=C sort
bash examples/reference-certified-world-v1/operator/verify.sh examples/reference-certified-world-v1
sed -n '1,120p' examples/reference-certified-world-v1/certification/replay-report.txt
sed -n '1,120p' examples/reference-certified-world-v1/certification/root-integrity-report.txt
sed -n '1,120p' examples/reference-certified-world-v1/genesis/world-roots.toml
sha256sum /tmp/everarcade-v0.1-world.evr
```

These commands build `world.evr`, list package contents, verify certification, inspect replay evidence, inspect root evidence, and print the package hash.
