# What Can I Build?

EverArcade is intended for deterministic worlds that benefit from declared rules, reusable gameplay capabilities, and verifiable continuity.

## Arena World

- **RustRig combinations:** identity, movement, combat, inventory, loot, continuity.
- **Continuity model:** short epochs, match receipts, replayable combat outcomes, reset or seasonal lineage.
- **Gameplay loop:** enter arena, move, attack, collect rewards, emit receipts, replay match history.

## Frontier World

- **RustRig combinations:** identity, movement, resources, structures, crafting, inventory, continuity.
- **Continuity model:** growing map history, persistent resource extraction, checkpointed settlement progress.
- **Gameplay loop:** explore, harvest, build, defend, craft upgrades, preserve discovered territory.

## Settlement World

- **RustRig combinations:** identity, inventory, market, governance, structures, resources, continuity.
- **Continuity model:** economic receipts, governance votes, checkpointed settlement state, migration-ready public records.
- **Gameplay loop:** gather resources, trade, vote on upgrades, expand shared infrastructure.

## Social World

- **RustRig combinations:** identity, permissions, reputation, governance, inventory, continuity.
- **Continuity model:** durable membership history, social graph snapshots, moderation/governance receipts.
- **Gameplay loop:** join, form groups, earn reputation, create events, govern shared spaces.

## Civilization World

- **RustRig combinations:** identity, movement, resources, structures, crafting, market, governance, combat, quests, continuity.
- **Continuity model:** long-running epochs, lineage-preserving migrations, replay windows, restore checkpoints.
- **Gameplay loop:** found settlements, allocate labor, trade, research, govern, compete, recover after failures.

## How to choose

Start small. Pick the template closest to the first visible player action, then add RustRigs only when the World Contract can explain the invariant they preserve.
