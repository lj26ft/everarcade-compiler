# Trading World Guide

## Create

```bash
everarcade new my-trading --template trading
```

## Build

```bash
everarcade build --project my-trading
```

## Deploy

```bash
everarcade deploy --project my-trading --target local
```

## Customize

Start with `src/game.js`, then add systems, assets, logic, and content without changing the template foundation.

## Included Systems
- Assets
- Inventory
- Marketplace
- Economy
- Vault Integration

## Demonstrates
- Creator Monetization
- Asset Ownership
- Settlement Flow
