# Civilization Guide

## Create

```bash
everarcade new my-civilization --template civilization
```

## Build

```bash
everarcade build --project my-civilization
```

## Deploy

```bash
everarcade deploy --project my-civilization --target local
```

## Customize

Start with `src/game.js`, then add systems, assets, logic, and content without changing the template foundation.

## Included Systems
- Settlements
- Citizens
- Governance
- Economy
- Regions

- Civilization Runtime
