# Sandbox Guide

## Create

```bash
everarcade new my-sandbox --template sandbox
```

## Build

```bash
everarcade build --project my-sandbox
```

## Deploy

```bash
everarcade deploy --project my-sandbox --target local
```

## Customize

Start with `src/game.js`, then add systems, assets, logic, and content without changing the template foundation.

## Included Systems
- World
- Inventory
- Marketplace
- Economy
- Events

## Demonstrates
- Minimal Assumptions
- Maximum Flexibility
