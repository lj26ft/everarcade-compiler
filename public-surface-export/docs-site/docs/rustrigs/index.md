# RustRigs

RustRigs are reusable gameplay mutation libraries and domain modules. They are useful for authoring, examples, and future world contracts, but should be treated as alpha unless explicitly classified otherwise in `MATURITY.md`.

Subsystem READMEs live in `rustrigs/*/README.md` for domains such as combat, inventory, economy, crafting, factions, movement, quests, and world simulation.
