# Settlement Boundary

## Must Eventually Settle
- ownership
- economy
- vault continuity
- disputes
- governance

## Does Not Require Settlement
- frame updates
- local simulation
- AI ticks
- transient combat state
- ephemeral world events

## Boundary Principle
Deterministic local execution is the primary runtime path. Settlement is deferred to epoch-level continuity proofs and applies only to sovereign state transitions that must be externally final.
