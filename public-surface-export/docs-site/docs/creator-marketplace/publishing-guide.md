# Publishing Guide

Publish records are replayable intent records. A creator publishes a versioned component, assigns licensing metadata, and allows other creators to browse, install, import, or depend on it through SDK and portal surfaces.

Marketplace v0.1 creates publication records, usage records, royalty records, version records, discovery records, analytics records, and replay records. It avoids protocol changes and does not create real payments.

Portal actions covered in v0.1 are Publish, Browse, Install, Update, and Deprecate.
