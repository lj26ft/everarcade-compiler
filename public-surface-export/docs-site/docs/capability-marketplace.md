# Capability Marketplace

The Capability Marketplace is where worlds discover reusable capabilities: gameplay systems, governance modules, operator tools, proof helpers, and treasury integrations.

Each capability listing should show name, author, version, worlds using it, reward model, and verification status. Worlds install capabilities because they are useful, but they should expose capability status so players and operators understand what is trusted, verified, or experimental.

[Browse Worlds](./worlds.md) to see capabilities in context.
