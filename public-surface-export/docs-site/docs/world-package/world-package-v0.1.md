# EverArcade World Package v0.1

The world is the sovereign object. A `world.evr` package is portable continuity: contract, runtime metadata, genesis state, replay/receipt/state roots, continuity root, schemas, assets, and proofs.

The package is deterministic: files are ordered lexicographically, JSON files are emitted canonically by the CLI, and the archive hash is computed over the exact bytes of the `EVRWORLD` archive.

The lease is temporary infrastructure and never becomes world identity.
