# Game Discovery Player Guide

The Game Discovery Network helps players browse sovereign EverArcade games without changing game runtime state or settlement authority.

## Flow

1. Browse featured, recent, popular, or category lists.
2. Open a game detail record with creator, version, status, category, and tags.
3. Install through the Player Gateway launcher.
4. Launch through the Player Gateway launcher.
5. Replay records can verify catalog, search, install, launch, and analytics roots.

Discovery is an index and recommendation layer only; games remain owned by their creators and deployments.
