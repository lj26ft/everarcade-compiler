# Quick Start

1. Create: `everarcade new --template arena --name my-arena`
2. Build: `everarcade build --project my-arena`
3. Test: `everarcade test --project my-arena`
4. Deploy: `everarcade deploy --project my-arena --target local`
5. Publish: `everarcade publish --project my-arena --channel creator-testnet`

Target outcome: first running game in under one hour.
