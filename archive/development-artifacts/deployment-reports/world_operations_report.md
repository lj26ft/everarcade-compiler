# World Operations Report

## Status
A single operations dashboard exposes online players, world health, simulation health, runtime health, replay health, deployment health, and live runtime metrics.

## Administration
World settings, player management, runtime controls, deployment controls, and rollback controls are recorded as deterministic admin receipts.

## Validation
Covered by `test_runtime_monitoring_equivalence`, `test_live_world_operations`, and Studio dashboard tests.
