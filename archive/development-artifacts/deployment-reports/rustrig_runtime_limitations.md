# Rustrig Runtime Limitations

Status: Partial

The deterministic runtime surface is executable for primitive protocol records. Renderer, history, and federation domains remain scaffold-level runtime domains by project instruction. Intent records are intentionally non-authoritative and require external orchestrators for deployment/XRPL execution.
